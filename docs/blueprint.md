# **App Name**: Qirat Academy

## Core Features:

- Live Navigation: Sticky navigation bar displaying the 'Qirat' brand logo, curriculum links, teacher directory, parent portal access, and a prominent 'Check Batch Availability' call-to-action.
- Dynamic Hero Section: Engaging hero section with a compelling headline and sub-headline, a large primary call-to-action for 'Start Child Assessment & Enroll', and a trust badge emphasizing low teacher-to-student ratio and pricing.
- Scarcity & Quality Control Explainer: Dedicated section detailing why batch sizes are limited, explaining Qirat's unique interactive platform features and commitment to maintaining premium educational standards.
- Interactive Assessment Gateway: An interactive, app-like UI card initiating 'Step 1' of the diagnostic quiz by asking the child's age, providing clickable buttons for age range selection.
- Personalized Learning Path Tool: After a child's age is selected in the assessment gateway, an AI tool will generate a recommended personalized learning path overview based on the input.
- Feature Grid Showcase: A 3-column feature grid presenting key platform benefits with clean, minimalist SVG icons, such as 'Certified Aalims', 'Holistic Curriculum', and 'Parent Portal'.
- Mission-Driven Footer: A dark-themed footer featuring Qirat's mission statement, 'Assalamu Alaikum', and standard navigation links like Privacy, Terms, and Contact Us.

## Style Guidelines:

- Primary brand identity color: Trustworthy Forest Green (#064E3B), representing growth, nature, and the brand's heritage.
- Background color: A serene and understated Off-White/Green-Tinted Gray (#F0F4F3), offering a clean and expansive canvas with ample whitespace.
- Accent color for Call-to-Action: Warm, premium Gold (#F59E0B), strictly used for primary interactive elements to signify value and urgency.
- Text color: Stark Black (#000000) for high contrast and readability across all textual content.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for all content as per user preference and the Y-Combinator aesthetic.
- Clean, minimalist SVG icons to complement the 3-column feature grid, ensuring visual clarity and a contemporary feel.
- Layout emphasizes immense whitespace and a strong visual hierarchy, guiding the user's eye naturally through the content, with a sticky navigation bar and a centrally floated, interactive card for the assessment.
